    il peut donner :
        - Une bases de donnees
        - Authentification (login/register)
        - API automatique
        - stockage de fichier
        - temp reel (chat, notification)

PRISMA : 
    Un outil pour parler a la base de donnees facilement.

    Sans sql, au lieu de faire :
        SELECT * FROM users WHERE id = 1;
    on fait tout simplement :
        prisma.users.findUniques({where: {id : 1} })

    Donc prisma : Simplifie les requetes a la base.

    const user = await prisma.user.findMany();
